# CollabCode  
### Version Control Web App

## Project Overview
CollabCode is a web-based version control platform that allows users to create, share, and collaborate on coding projects.

It includes project management, activity tracking, file sharing, user connections, and a check-in/check-out system.


---

## Features
- User registration and login
- User profiles and friend connections
- Create and manage coding projects
- Upload and share project files
- Check-in and check-out system
- Local and global activity feeds
- Project discussions
- Search for users, projects, and hashtags
- Admin controls
